# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] — April 2026

### Added
- Initial release of AI Risk Assessment Framework v1.0
- 6 risk domains with 46 assessment questions
- Excel toolkit with 4 sheets (Instructions, Checklist, Dashboard, Summary)
- PDF report generator (Python/ReportLab)
- Sample assessment: ChatGPT (GPT-4) — HIGH risk (3.72/5.0)
- Framework alignment documentation (NIST AI RMF, ISO/IEC 42001, OWASP LLM Top 10, EU AI Act)
- Domain definitions, scoring methodology, and controls library

### Framework Coverage
- D1: Bias & Fairness (7 questions, 15% weight)
- D2: Data Privacy & Protection (8 questions, 20% weight)
- D3: Model Transparency & Explainability (7 questions, 10% weight)
- D4: Security Risks (9 questions, 25% weight)
- D5: Accountability & Governance (8 questions, 15% weight)
- D6: Compliance & Legal Risk (7 questions, 15% weight)
